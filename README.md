# Leer-el-c-lculo.app
Leer el Cálculo es una aplicación educativa que invita a comprender el cálculo desde la lectura y la escritura. Inspirada en La lectura del Cálculo de Bernardo Mayorga, propone pensar conceptos como el límite a través del lenguaje, la reflexión y el sentido, más allá de los procedimientos.
